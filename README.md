# Sudoku (DIO) —  Dificuldade 

Este repositório é um **fork** do projeto/desafio **Sudoku** da **Digital Innovation One (DIO)**.
O projeto original permite iniciar um tabuleiro via argumentos e jogar via **console** ou **interface gráfica**.

## Alterações deste fork
- Seleção de dificuldade (**Fácil / Médio / Difícil**) ao iniciar a **GUI** (popup) e também no **modo console**.
- 3 tabuleiros prontos 
- Botão **“Novo jogo”** na GUI para escolher a dificuldade e reiniciar o tabuleiro.

## Como executar
**Requisitos:** Java 11+ 

Projeto Java simples (**sem Maven/Gradle**).

- **GUI:** execute `br.com.dio.UIMain`
- **Console:** execute `br.com.dio.Main`
